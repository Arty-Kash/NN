# NN
A sample program to visualize Neural Netwoak training process.


## Codespacesでの実行方法
ターミナルで、下記コマンドを実行し、ブラウザを開く
- bash
- pip install -r requirements.txt
- python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000
  - -m uvicorn ： Pythonの中のuvicornというモジュールを起動
  - main:app ： main.py というファイルの app(=FastAPI()) というインスタンスを実行するという意味。
  - --reload ： コードを書き換えると自動でサーバーが再起動します（開発に便利）。
  - --host 0.0.0.0 ： Codespacesの外部（ブラウザ）からアクセス可能にするために必要です。
